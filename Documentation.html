Configuracion del entorno de desarrollo


1 paso: Instalación de XAMPP

Se deberá descargar desde la versión ofical de la página, consultar: apachefriends.org

2 paso: Verificar que funciona Apache

Usalmente siempre funciona el servidor de apache una vez instalado, si no es el caso verificar en donde se realizó la instalación y verificar la documetación ofical de la págiona.

3 paso: Abrir el archivo Htdocs

Dirigirse al disco de instalación, despues a la capeta Xampp , htdocs y finalmente el proyecto se crea dentro de esa carpeta. (C -> Xampp -> Htdocs -> "tu archivo")

4 paso: Crear una carpeta con nombre de preferencia

Dentro de la carpeta se debe crear nuestro archivo usalmente una carpeta para almencar nuestros demás archivos del proyecto

5 paso: Crear un archivo con extención ".php"

El archivo deberá tener la extención php y usalmente se llamará como (main ) o (index)

6 paso: Abrir en el localhost nuestro archivo

Finalmente abrimos nuestro archivo en el servidor local de la máquina, para verificar que realmente todo está correctamente instalado.
